<span class=" text-lg">Install and configure Easy UI for Next.js</span>

#### 1. Information

This project and all templates are written in TypeScript. We recommend using <b>TypeScript</b> for your project as well.

#### 2. Download the template

Start by downloading a new Easy UI template.

#### 3. Install dependencies:
   ```bash
   pnpm install
   ```

#### 4. Run the development server:
   ```bash
   pnpm dev
   ```

#### 5. Open this in your browser to see the result.

 ```bash
   http://localhost:3000
   ```
